# 🏥 HealthPrime

Uma plataforma web institucional moderna e responsiva desenvolvida para o setor de saúde e bem-estar. O projeto apresenta de forma clara os serviços médicos oferecidos, especialidades disponíveis, diferenciais da clínica e canais para agendamento de consultas.

---

## 🚀 Tecnologias Utilizadas

* **HTML5** (Estrutura semântica e focada em acessibilidade)
* **Tailwind CSS** (Estilização limpa, moderna e totalmente responsiva)

---

## ✨ Funcionalidades

* 🩺 **Apresentação de Especialidades:** Área dedicada para exibir de forma organizada os serviços médicos e tratamentos disponíveis.
* 👨‍⚕️ **Corpo Clínico:** Seção estruturada para apresentar a equipe de profissionais e especialistas.
* 📅 **Direcionamento para Agendamentos:** Botões e chamadas para ação (CTAs) estratégicas para facilitar a marcação de consultas.
* 📱 **Design Responsivo:** Interface otimizada para uma navegação fluida em smartphones, tablets e computadores.
